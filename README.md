# Any Project Template
**Шаблон git-репозитория для нового проекта**

## Особенности
1. Директория [Fuzz](./Fuzz/README.md) содержит преднастроенный функционал для автоматизации процесса разработки.
2. Директория [WiKi](./WiKi/README.md) предназначена для агрегации документации о новом проекте.

## Начало работы
1. Выкачать данный репозиторий в папку с соответствующим названием.
2. Удалить `.git` и инициализировать новый `git init`.
